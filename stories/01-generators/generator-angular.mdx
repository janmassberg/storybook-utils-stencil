# Angular Source Generators

## Angular Template

Use `generateSourceAngularHtml` to generate the Angular template source.

<Source
    code={`
import { generateSourceAngularHtml } from "@jmassberg/storybook-utils";\n
const angularHtmlSource = generateSourceAngularHtml(${generatorIntegrationSourceArgs});
`}
/>

### Output

<Source
    dark
    language="html"
    type="code"
    code={generateSourceAngularHtml(generatorIntegrationArgs)}
/>
