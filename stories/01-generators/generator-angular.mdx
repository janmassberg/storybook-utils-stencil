# Angular Source Generators

## Angular Template

Use `generateSourceCodeAngularHtml` to generate the Angular template source.

<Source
    code={`
import { generateSourceCodeAngularHtml } from "@jmassberg/storybook-utils";\n
const angularHtmlSource = generateSourceCodeAngularHtml(${generatorIntegrationSourceArgs});
`}
/>

### Output

<Source
    dark
    language="html"
    type="code"
    code={generateSourceCodeAngularHtml(generatorIntegrationArgs)}
/>
