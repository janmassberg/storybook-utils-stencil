# HTML Source Generators

## HTML Markup

Use `generateSourceHtmlMarkup` to generate the HTML source.

<Source
    code={`
import { generateSourceHtmlMarkup } from "@jmassberg/storybook-utils";\n
const htmlSource = generateSourceHtmlMarkup(${generatorIntegrationSourceArgs});
`}
/>

### Output

<Source
    dark
    language="html"
    type="code"
    code={generateSourceHtmlMarkup(generatorIntegrationArgs)}
/>

## HTML Script Tag

Use `generateSourceHtmlScript` to generate the JS script source.

<Source
    code={`
import { generateSourceHtmlScript } from "@jmassberg/storybook-utils";\n
const jsSource = generateSourceHtmlScript(${generatorIntegrationSourceArgs});
`}
/>

### Output

<Source
    dark
    language="html"
    code={generateSourceHtmlScript(generatorIntegrationArgs)}
/>
