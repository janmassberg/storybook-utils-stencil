# HTML Source Code Generator

## HTML

Use `generateSourceCodeHtml` to generate the HTML source.

<Source
    code={`
import { generateSourceCodeHtml } from "@jmassberg/storybook-utils";\n
const htmlSource = generateSourceCodeHtml(${generatorIntegrationSourceArgs});
`}
/>
